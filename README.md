# Hi, I'm building practical AI-agent automation

I turn recurring operational pain into small, local-first open-source tools. Current focus: making unattended coding agents reliable enough to run on a schedule.

## Featured project

### [AgentCron](https://github.com/shkyyy18/cc-autopilot)

**Cron + watchdog for Codex, Claude Code, Gemini CLI, and custom commands.**

- timeout and process-tree cleanup
- silent-failure detection
- bounded retries and structured UTF-8 logs
- Windows Task Scheduler plus Linux/macOS cron
- zero runtime dependencies
